# Participant Lifecycle

```text
Recruitment
→ Application
→ Eligibility Review
→ Interview
→ Background Review
→ Acceptance
→ Program Enrollment
→ Onboarding
→ Training
→ Ready for Matching
→ Placement Match
→ Placement Onboarding
→ Active Employment
→ Performance and Development
→ Career Transition
→ Permanent Employment or Other Outcome
→ Alumni / Follow-up
```

## UX principle

The participant sees a simplified journey and a clear current step, not the full internal operational complexity.

The Training step is the three-program portable readiness gate in `ADR-017`. The later Placement Onboarding step covers the actual host site, hazards, equipment, procedures, and assigned-task competency; the two gates remain independent.
