# Epic 2 — Participant Applications

## Goal
Allow applicants to apply and track progress.

## Stories
1. Public How It Works page.
2. Applicant account onboarding.
3. Create and save draft application.
4. Upload required documents.
5. Submit application.
6. View participant-safe application journey.
7. Operations applications queue.
8. Eligibility review.
9. Interview workflow.
10. Background decision workflow.
11. Acceptance and rejection.
