# User Flows

## Applicant

Learn → Check expectations → Create account → Application → Documents → Submit → Review stages → Decision

## Accepted participant

Acceptance → Onboarding → Training → Ready for Matching → Proposal decision → Placement onboarding → Active placement → Hours → Career transition

## Shelter

Sign in → Review proposed placement → Approve/request change/decline → View assigned participant → Approve hours → Evaluate → Report incident

## Operations

Dashboard queue → Application review → Eligibility → Interview → Background → Accept → Enrollment → Training → Match → Placement → Activate → Monitor → Complete
