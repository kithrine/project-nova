# Security and Privacy

## Classification

- Public
- Internal operational
- Confidential personal
- Restricted case information
- Highly restricted

## Highly restricted examples

- Background reports
- Criminal-history details
- Government ID documents
- Serious incident investigations
- Sensitive audit records

## Controls

- Least privilege
- Organization scope
- Restricted queries
- Audit events
- Named exports
- No restricted global search
- No sensitive logging
- Secure object storage
- Environment separation

## Legal review required

- Retention periods
- Background-check obligations
- Employment records
- Grant data requirements
- Participant access and correction rights
