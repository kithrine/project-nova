- Initialize Next.js, TypeScript, Tailwind, DaisyUI
- Configure native CSS conventions
- Connect GitHub, Vercel, Clerk, Neon, and Prisma
- Establish environment variables
- Configure lint, type checks, Vitest, Playwright, and CI
- Create base app shells and design tokens
