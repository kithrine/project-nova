# Project Nova Blueprint

## Vision

Build the operating system for grant-funded transitional work, beginning with animal shelters.

## Strategic position

Project Nova is a workforce intermediary, not merely a staffing service or software platform. It coordinates participants, host worksites, training, funders, and employment outcomes.

## Mission center

The participant’s journey toward sustainable employment.

## Operational center

The Placement.

## Product model

- Public recruitment and explanation
- Guided application experience
- Nova Operations case-management workspace
- Shelter partner supervision workspace
- Auditable reporting foundation

## MVP goal

Operate one real pilot cohort safely and coherently.

## Product phases

- V1: Run the pilot
- V2: Improve communication, reporting, and program administration
- V3: Scale through integrations, automation, and blended funding

## Technical thesis

Use one full-stack Next.js application with modular domain boundaries rather than premature microservices.

## UX thesis

Every experience should answer one question:

- Participant: What do I do next?
- Shelter: Who needs my attention?
- Operations: What work is waiting?
